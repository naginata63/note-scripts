AIに競合14チャンネルを丸裸にさせた話 — 量産型は負けていた

---

ゲーム実況の切り抜きチャンネルを始めた。

同じジャンルに、公認の切り抜きチャンネルが13個ある。

**勝てるのか？** まず感情より先に、データで全部調べることにした。

---

## 第1章：14チャンネル全部並べてみた

YouTube Data APIを使って、競合13チャンネル＋自分のチャンネルの統計を一括取得した。スクリプト1本、数分で完了。登録者数・総再生数・動画本数がすべて揃った。

そこで「1本あたり再生数」を計算して並べてみた。

結果がこれ。

- **A（アルバム型）**：653本 → 1本あたり **13万再生**
- **B（シアター型）**：203本 → 1本あたり **8万再生**
- **C（老舗最大手）**：1,458本 → 1本あたり **6.2万再生**
- **D（小話型）**：559本 → 1本あたり **5.7万再生**
- **E（ROOM型）**：870本 → 1本あたり **4.3万再生**
- **F（ニッチ特化型）**：355本 → 1本あたり **4.0万再生**
- **G（ショート専門）**：1,088本 → 1本あたり **3.8万再生**
- **H（放送型）**：461本 → 1本あたり **3.5万再生**
- **I（ポケット型）**：643本 → 1本あたり **3.3万再生**
- **J（特化型）**：674本 → 1本あたり **2.1万再生**
- **K（ファン向け）**：1,115本 → 1本あたり **2.1万再生**
- **L（量産型）**：1,681本 → 1本あたり **1.3万再生**
- **M（超量産型）**：2,303本 → 1本あたり **8千再生**
- **自分**：31本 → 1本あたり **503再生**

Aは653本で13万/本。Mは2,303本で8千/本。**本数は3.5倍なのに、1本あたりの効率は16分の1**。

量産すれば伸びるわけじゃない。じゃあ何が違うのか。

---

## 第2章：AIが見つけたパターン

上位・下位チャンネルの特徴をAIに分析させた。見えてきたのは、シンプルな構造だった。

**上位チャンネルの共通点**

- A（13万/本）：複数シーンを1本にまとめる「アルバム形式」。元動画を切り出すだけでなく、**構成という編集付加価値**がある
- B（8万/本）：感動・ドラマ系のシーンに完全特化。テーマを絞ることで**刺さる視聴者に深く刺さっている**
- C（6.2万/本）：2021年からの老舗。累計9,000万再生超。新規が同じ戦略を取っても、蓄積の差で勝てない

**下位チャンネルの共通点**

- L（1.3万/本）：月32本の超高頻度投稿。全メンバー・全シリーズをカバーする総合型。**広く浅く速く**が効率を下げている
- M（8千/本）：2,303本で全チャンネル中最多投稿。**本数は最大なのに、効率は最低**

ショート専門（G）は1,088本で3.8万/本。中位だった。ショートだけでは飛び抜けない。

続けてサムネを収集してAIで傾向を分析した。

**サムネ分析（317枚）で見えたこと**

- 上位チャンネル：キャラクターが画面の80〜90%を占める。煽りテキストが大きい
- 自分のサムネ：キャラが画面の約68%。競合と並べると埋もれる

数字で見ると「ああそうか」となる。感覚でやっていたら気づかなかった。

---

## 第3章：じゃあどう戦うか

データが揃ったところで、自分の戦略を考えた。

大手（C）に真正面からぶつかるのは無謀だ。5年分の蓄積には勝てない。でも、量産型（L、M）と同じことをやっても効率が悪い。

AIを使うと、何が変わるか。

**手動チャンネルが数日かけることを、AIパイプラインなら数時間でやれる**

元動画が公開されてから競合が投稿するまで、翌日から数日かかることが多い。AIパイプラインなら当日中に投稿できる。速さが差別化になる。

さらに「速さだけ」でなく「質」も維持できるのがポイントだ。

- **ショート動画**でバイラル狙い → 早い段階でチャンネルを認知させる
- **ハイライト（8分超）**で再生時間を稼ぎ広告収益につなげる

量産型の「速さだけ」でも、シアター型の「質だけ」でもない。両方できるのが、AIを使うアドバンテージだと考えた。

---

## おわりに

14チャンネルのデータ取得・分析・戦略策定まで、かかった時間は**1日**。

感覚で「なんとなく勝てないかも」と思っていたものが、数字になった。数字になると「どこで勝てるか」が見える。

YouTubeに限らず、「同業者を全部並べて数字で比較する」のは最強の初手だと思う。それがAPIとAIで、費用ほぼゼロ・1日で完結する時代になっている。

チャンネルはまだ31本・登録者23人。始まったばかりだ。でも、どこを狙うかは決まった。
